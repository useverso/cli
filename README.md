# VERSO CLI

CLI tool for initializing and managing [VERSO](https://github.com/useverso/verso) projects.

> **Status:** Under development. Not yet published to NPM.

## Planned Usage

```bash
# Initialize VERSO in your project
npx useverso init
```

The `init` command will:

1. Copy the `.verso/` template files into your project
2. Walk you through basic configuration (scale, autonomy levels, board provider)
3. Set up your project board columns if using GitHub Projects

## Development

```bash
# Install dependencies
npm install

# Run locally
npm run dev

# Build
npm run build
```
